Back-End nc-games-API

a RESTful API for nc-games

This project served as a RESTful API back-end to my front-end nc-games project. I completed as part of my Full Stack Developer course from Northcoders

Here is a link to the website

Here is a link to the Github repo

I enjoyed this project as I feel I learned a great deal of concepts while at the same time applying data manipulations skills that I had picked up from practicing daily katas similar to those you would find on Code Wars.

Some things I utilised in this project include: PostgreSQL queries (including JOINs and aggregate functions), express, routing, extensive testing with Jest Supertest, custom TDD util functions and creating reference objects .

I was also introduced to: the MVC architectural pattern, CICD, environment variables and custom error handling responses
